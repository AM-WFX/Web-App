How to Use This Lab

This lab accepts a wide variety of valid CSS selectors. Your goal is to find a selector that is precise enough to select only the target element. For example, if the target has an ID of #normal-selector, many answers are valid:

  • ID Selector: #normal-selector
  • Attribute Selector: [id="normal-selector"]
  • Partial Attribute Selectors: [id*="normal"] (contains), [id^="norm"] (starts with), or [id$="lector"] (ends with)

The key is precision. If your selector (like div or .common-class) matches other elements in the box, it will fail. Good luck!
